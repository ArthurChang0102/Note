# pip

* Q: 什麼是 pip 呢？

  A: pip 是一個以 Python 寫成的軟體包管理系統，他可以安裝和管理軟體包

* 使用方式

  * 安裝：install

    * `pip install [套件名稱]`

  * 移除：uninstall

    * `pip uninstall [套件名稱]`

  * 列出所有安裝的套件：list

    * `pip list`

  * 匯出套件清單：freeze

    * `pip freeze > ./requirements.txt`

      > 這是指將所有的套件名稱與版本寫到 requirements.txt 這個檔案中

* 裝個套件吧～

  * `pip install requests` 安裝一下我們等等會用到的爬蟲套件
