# dd


### make Linux install usb
* command: `dd if=[iso-file-path] of=[usb] bs=[size (4M)]`
  > if = input  
  > of = output  
  > bs = block size


### delete Linux install usb
* command: `dd count=2 if=/dev/zero of=[usb]`
